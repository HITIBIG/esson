# ESSONYOGA - Professional Yoga & Fitness Supply Chain

[![Site Status](https://img.shields.io/website?url=https%3A%2F%2Fhitibig.github.io%2Fesson%2F)](https://hitibig.github.io/esson/)

**ESSONYOGA** 是一家位于义乌的专业 B2B 瑜伽与健身器材供应商。本项目是 Essonyoga 的官方独立站展示窗口，旨在为全球健身品牌提供高标准、战略性的采购解决方案。

## 🚀 项目定位：从“制造”到“势能”

不同于传统的贸易型网页，ESSONYOGA 的设计理念融合了**势能营销 (Potential Energy Marketing)** 逻辑：
- **战略采购 (Strategic Sourcing)**：专注于解决 B2B 采购中的货期不确定性与质量波动痛点。
- **品牌赋能 (Brand Alignment)**：通过深度 OEM/ODM 咨询，确保产品材质与包装设计高度契合客户的品牌视觉体系。

## 📦 核心产品线

- **Anti-Burst Yoga Ball**: 工业级蜂窝防爆结构，SGS 认证，适配高强度商业环境。
- **Eco-Friendly Yoga Mat**: 100% 可回收 TPE/天然橡胶材质，专注于防滑性能与环保溢价。
- **Multi-Resistance Bands**: 100% 天然乳胶，抗拉断技术，支持私有化标签定制。

## 🛠️ 技术实现方案

本项目采用了极致简洁且高性能的技术栈，确保海外买家在弱网环境下也能秒开页面：

- **Frontend**: Tailwind CSS 3.0 (响应式布局、极简主义 UI)
- **Icons**: Font Awesome 6.0
- **Interaction**: 原生 JavaScript 驱动的图片切换矩阵与淡入切换效果
- **Lead Generation**: 集成 **Tally.so** 深度表单，支持 Webhooks 实时同步询盘至飞书/邮件
- **Data Analytics**: 部署 **Google Analytics 4 (GA4)** 实现全链路行为追踪
